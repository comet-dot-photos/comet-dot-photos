Comet.Photos Quickstart

What is this?

Comet.photos is an open source application for filtering and viewing images taken by the OSIRIS Narrow Angle Camera on board the European Space Agencyâ€™s Rosetta mission to Comet 67P/Churyumov-Gerasimenko. People interested in checking out the program can run it in a web browser by simply visiting https://comet.photos (no software download required!), but scientists using Comet.photos for their research can install it locally on their computer for seemingly instantaneous performance. The program is freely available to everyone interested in comets and space exploration. Using Comet.photos, you can manipulate a 3D model of Comet 67P, paint a region of interest, and quickly retrieve all images taken of that region. You can further filter down the image set by specifying other image parameters, and at any time you can navigate through all of the matching images, and display them in either two dimensions or texture-mapped onto the 3D comet model. This is done in 4 easy steps.

Step 1: Finding the Region of Interest

In the center of the screen is a 3D model of Comet 67P. Rotate the comet about its center point by using the left mouse button, or move the comet with the right mouse button. There are three ways to zoom in or out on the comet: by using your mouse's scroll wheel, by moving the mouse up or down with the middle button pressed, or by using a pinching gesture on a trackpad. Rotate and scale the 3D model view in this way until the desired region of interest is visible.

Step 2: Painting the Region of Interest

The user indicates the region of interest by painting on the 3D model, with the help of the controls in the Paint Tools section of the control panel. First, enable painting mode by selecting the Enable Paint checkbox in the right hand control panel. While this check box is enabled, the mouse is used to paint a region of interest for retrieving images, rather than manipulating the comet 3D model. The cursor changes to a pink sphere, which is the 3D paint brush. The 3D paint brush size can be adjusted by either changing the Brush Size slider or accompanying numeric input field (which specify the brush size radius in meters), or by using the mouse's scroll wheel.

Once the brush size is set to the desired size, paint on the comet using the left mouse button. To erase, simply use the right mouse button instead. All of the paint can be cleared (erased) by clicking the Clear Paint button. The painted region of interest is used to geographically retrieve only images that include this region. To specify how much of the region of interest must be included in an image for it to be considered a match, adjust the Percent Overlap slider. After every paint stroke, the Matches text field updates to show how many of the images match the filter. To exit paint mode, deselect the Enable Paint checkbox. At any time, you can go back to paint mode to adjust the region of interest.

Step 3: Setting Other Image Filters

After specifying a region of interest, you may want to further filter down the image set by specifying additional photometric image parameters. This is done in the Image Filters section of the control panel on the right. For example, by adjusting the Meters per Pixel slider, you can indicate a maximum or minimum number of meters per pixel for an image to match. Generally speaking, by reducing this number, you will retrieve images taken closer to the comet. The Emission Angle slider indicates the maximum and minimum angles between the surface normal and the spacecraft when taking an image. Reduce this to retrieve only images that appear more "head on" or increase this to view the region at at a more obtuse angle. The Incidence Angle slider indicates the maximum and minimum angles between the surface normal and the sun. Reducing this will tend to reduce shadows, and increasing it will show more shadows. The Phase Angle slider indicates the maximum and minimum angles between the sun and the spacecraft, taken at the region of interest. Reducing this will also result in less of the selected region being in shadow, but can also select for certain lighting effects. As these additional search parameters are adjusted, the Matches field will continue to update to show the number of images matching the photometric and spatial filters.

Step 4: Navigating through the Images

Once the set of more than 27,000 comet images has been filtered down to a more manageable quantity, you will likely want to start viewing them. You can do so with the controls in the Image Display and Navigation section of the control panel.

First, to start displaying images, choose a display option from the Show Image dropdown control. Initially this is set to "None", so no image is displayed. Choosing "Unmapped 2D" will display the image as taken from the Rosetta Narrow Angle Camera, in two dimensions. The last two choices project the image on to the 3D comet model. "Perspective" uses a perspective projection, and "Orthographic" uses an orthographic projection. While the Encircle Region checkbox is checked, a red circle is placed around the region of interest that the user had painted. This makes tracking a specific region much easier when displaying image after image. Note that if you navigate through the filtered images, projecting them onto the comet model, the texture will often appear distorted, because the image was taken from a different vantage point than the Rosetta orbiter camera. To see the view that the image was taken from, select the Spacecraft View checkbox. To see the square representing the field of view of the Rosetta Narrow Angle Camera extended to the comet, check the Show Viewport checkbox. Note that the viewport of the image may be at the periphery or outside of the current camera field of view, so you may have to zoom out (and in some cases rotate the comet) to see the blue square representing the viewport. To display standard coordinate axes for the comet, check the Axes checkbox.

The Image Index slider shows the index of the image in the filtered search results. You can adjust this slider to view other images. However, to simply move to the next image, it is easier to click on the Next Image button. Similarly, to go back to the previous image, click on the Previous Image button.

Since images in the filtered set are ordered by the time that they were taken, you will frequently several nearly identical images in a row. To quickly advance a day, month, or year, choose the appropriate time period from the Skip Duration dropdown box. Then use the Skip Forward or Skip Backward buttons below it to advance to the next or previous image that was taken at least the specified skip duration from the current image.

As you navigate through the filtered image set, it is helpful to look at the text fields in the Image Data section of the control panel. The Matches field shows how many images match the spatial and photometric filters, and have been returned by the search, as well as the total number of images. The File Name field shows the name of the current image's file. Scientists interested in doing additional research on an interesting image will likely want to take note of its file name. The Time field shows when image was taken (in ANSI standard, UTC time format). The Image Info field specifies the index of the image in the entire data set (#), the meters per pixel of the currently selected image (m), and if a region of interest is painted, the emission (e), incidence (i), and phase (p) angles in degrees. If you would like to download a file containing the set of image file names (mostly of use to comet scientists), click on the Download File Names button.
